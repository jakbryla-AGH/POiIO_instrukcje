<h1> 2. Kubeczki [C++/CLI] </h1>
   
Instrukcje prowadzą przez naukę tworzenia aplikacji okienkowych w języku C++/CLI. Możesz zapoznać się z działaniem aplikacji poprzez ściągnięcie pliku setup.exe i instalację programu "kubeczki".

<h2> Zawartość </h2>
Poszczególne lekcje umieszczone są w folderach, wykonuj je w kolejności:</br>
1. Wprowadzenie</br>
2. Dodawanie kubeczków</br>
3. Dodawanie substancji</br>
4. Wykresy</br>
5. Plik instalacyjny</br>
</br>
W każdym folderze najdziesz instrukcję w formacie *.pdf oraz ewentualne dodatkowe pliki konieczne do wykonania zadań.
